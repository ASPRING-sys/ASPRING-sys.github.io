King:

王是代码实现难度较大的棋子之一,因为他虽然看似只是一次移动一格,但是由于他是整个棋盘上最重要的棋子,他一旦被攻击就必须要做出行动为他化解危机,而且还有与车的联动:王车易位!并且关于易位也有一套特殊的规则,就是王的移动路径以及其本身所在的格子不能被攻击

Pawn:

接下来就是最难的棋子,兵.兵的行动规则是:
1.每个兵的第一步可以走1个单位或者2个单位,此后就只能往前走一个单位
2.兵只能攻击斜前方的两个格子
3.兵走到对方底线时能够升变为除了兵之外的任何一个棋子
4.当敌方兵一次性走两格与我方兵并排时,我方兵可以选择将其吃掉,并移动到它后方(吃过路兵),且该规则只在这一回合内生效

如何搭建自己的博客网站

第一步:准备底层工具 Node.js Git

1.打开电脑终端(win+R:cmd)
2.检查自己是否已经下载了Node.js和Git:
node -v
git –version
3.若第二步未得到版本号,则执行以下命令进行下载:
winget install OpenJS.NodeJS.LTS
winget install Git.Git
注:安装完成后请关闭该终端重新打开才会生效

第二步:安装框架并初始化本地项目

1.安装Hexo命令行工具:
npm install -g hexo-cli
2.创建一个打算存放博客的目录,如My_Blog
3.进入终端,进入刚才创建的目录:
cd My_Blog
注:如果该目录创建在C盘之外的盘,先输入D:切换盘符
4.依次执行以下命令:
hexo init my_blog
cd my_blog
npm install
5.输入以下命令将项目跑起来:
hexo server
6.打开浏览器访问以下网址:
http://localhost:4000

第三步:连通 GitHub

1.如果你刚才终端里的 hexo server 还在运行，请在终端里按下 Ctrl + C 中止它
2.登陆你的github,新建一个仓库,将其命名为:
你的GitHub用户名.github.io
3.在你的 my_blog 终端目录下，运行以下命令:
npm install hexo-deployer-git –save
4.请在你的 my_blog 文件夹根目录下，找到那个叫做 _config.yml 的文件
5.用代码编辑器打开它(记事本也行)
6.在文件的最底部找到deploy: 这一段并修改为:
deploy:
type: git
repo: (https://github.com/你的用户名/你的用户名.github.io.git)
branch: main
7.请在你的 my_blog 终端里，依次输入这三条指令(以后每次写完新文章也要执行):
hexo clean
hexo generate
hexo deploy
8.打开浏览器并访问https://你的GitHub用户名.github.io
注:
你以后写的文章都会在D:\My_Blog\my-blog\source_posts
这个文件夹里面,如my-first-post.md

总结:赶快开始搭建你的网站并发送第一篇博客吧!
